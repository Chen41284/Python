# VMTK
```
centerlines.py 显示PLY等网格模型中的中心线

extract_component.py  提取PNG等图像中的某一个通道并保存
extract_component_show.py 提取PNG等图像中的RGB三个通道并同时显示进行对比

imageFeature.py  计算特征图像以用于分割 vmtkImageFeatures
imageVesselEnhance.py 计算特征图像以用于分割 vmtkImageVesselEnhancement
imageObjectEnhance.py 计算特征图像以用于分割 vmtkImageObjectEnhancement

imageMorphology.py  将二进制或灰度形态过滤器应用于输入图像 vmtkImageMorphology
imagenNormalize.py 使用多个Otsu阈值自动标记图像 vmtkImageNormalize

imageVoiSelector.py  提取图像感兴趣的区域并保存
levelset_seg.py  图像的水平集分割，生成等值面

marchingCubes.py 移动立方体算法生成等值面
volume.py   体绘渲染
volume_extraction.py  3D图像的截取

read_image.py  显示CT等图像
view_CT_series.py 观察CT图像序列
view_tissues.py 查看图像序列中的多个器官组织
pngTovti.py  将png图像序列转化为vti等3D格式的文件

surface_capper.py 将网格的端口封住
surface_smooth.py 对网格的表面进行平滑
surface_clipper.py 对网格进行裁剪
surface_flow_extension.py 对网格的流进行拓展
surface_subdivision.py 对网格进行细分
```
